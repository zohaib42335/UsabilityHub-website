🚀 UsabilityHub — Remote User Research Platform

Validate your design decisions with real users — before you launch.
UsabilityHub is a remote user research platform that removes the guesswork from UX/UI design. Run tests, gather insights, and make smarter product decisions backed by real user feedback.

📌 Features
✅ Five-second tests to test first impressions

✅ Click tests to measure navigational expectations

✅ Surveys for structured user feedback

✅ Preference tests for A/B design comparisons

✅ Prototype tests for usability validation
